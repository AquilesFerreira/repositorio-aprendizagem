A Openeem BIOSCIENCE desenvolve produtos para o setor agrícola. Os produtos da Openeem BIOSCIENCE são 100% naturais, fabricados com tecnologia de ponta a partir de diferentes blends de extratos da Árvore do Neem, Azadirachta indica.

Concebidos para se inserirem no manejo nutricional e no manejo integrado de pragas (MIP), cada formulação foi pensada para oferecer flexibilidade de aplicação (solo, semente e foliar) e sinergia com outros produtos.

ESCOLHER OPENEEM FLEX É ESCOLHER PRODUZIR MAIS E MELHOR

Openeem FLEX é um potente fertilizante orgânico líquido de ação sistêmica que possibilita as plantas crescerem mais rápido e vigorosamente.

Openeem FLEX promove a saúde integral das plantas contribuindo para que alcancem o potencial genético produtivo máximo.

A BASE CERTA PARA SEU CULTIVO

Openeem MIX é um poderoso fertilizante orgânico de ação sistêmica e liberação lenta para aplicação via solo.

Openeem MIX promove com solidez o estabelecimento e desenvolvimento dos cultivos.

OPENEEM PLUS, O ALIADO DO PRODUTOR NO MANEJO INTEGRADO DE PRAGAS - MIP

Openeem PLUS é um adjuvante multifuncional indicado para adição em caldas de pulverização para potencializar aplicação de defensivos químicos e biológicos.

Openeem PLUS é um produto único que se insere no manejo integrado de pragas para maximizar resultados.
